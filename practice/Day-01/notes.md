# Day 1 Notes

## Five Terminal Commands

1. pwd - Shows the current working directory.
2. ls - Lists files and folders.
3. cd - Changes the current directory.
4. mkdir - Creates a new folder.
5. touch - Creates a new file.

## Three Git Commands

1. git init - Initializes a Git repository.
2. git add . - Stages changes for the next commit.
3. git commit -m "message" - Saves the staged changes.

## Git vs GitHub

Git is a version control system that tracks changes in files.
GitHub is an online platform used to store and share Git repositories.
